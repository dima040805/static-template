# Как переслать ссылку на чат с конкретным пользователем

{% note info %}

Вы можете легко переслать ссылку на чат с пользователем вашему менеджеру, чтобы он в один клик перешел к общению с клиентом в сервисе: урегулировал возражение или провел индивидуальную консультацию. Это особенно полезно, если нужно быстро ответить клиенту со смартфона.

{% endnote %}

Например:&#x20;

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FD1ndjSE7FIDVhkUYunJW%2Fimage.png?alt=media&#x26;token=c37fec6b-892a-450d-847a-1fa670553bcc" alt=""><figcaption><p>Пример заполнения блока "Отправить уведомления", чтобы отправить менеджеру ссылку на диалог с клиентом в разделе "Чаты"</p></figcaption></figure>

Чтобы автоматически создать и вставить в уведомление менеджеру ссылку на чат с нужным пользователем, выполним несколько простых шагов:

1. Заходим в раздел "Чаты"&#x20;
2. Открываем любой диалог
3. Копируем ссылку из адресной строки:

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FNrMIFAif79GpcDisLUX1%2Fimage.png?alt=media&#x26;token=757d464e-3a20-4421-8649-5b654869e8ec" alt=""><figcaption></figcaption></figure>

4. Последний шаг - просто заменяем в ней всё после `conversations/` на `%user_oid%,` где  `%user_oid%` - переменная, значение которой при отправке в боте заполнится id (уникальный идентификатор) пользователя, в диалог с которым нужно перейти. Таким образом бот автоматически подставит id нужного пользователя.&#x20;

{% note info %}

Обратите внимание: чтобы ссылка отправилась корректно, нужно предварительно делинковать ее в блоке "Отправить уведомление". Для этого просто выделаем всю ссылку и нажимаем сюда:

{% endnote %}
