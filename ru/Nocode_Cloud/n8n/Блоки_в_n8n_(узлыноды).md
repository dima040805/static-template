# Блоки в n8n (узлы/ноды)

## Что такое узел в n8n?

* **Узел** (или нода) — это элемент рабочего процесса, который выполняет конкретную задачу. Такой же, как и блоки в Smartbot Pro.
*   Узлы могут делать разные вещи, например:

    * Получать данные из API.
    * Отправлять сообщения.
    * Обрабатывать файлы и изображения.

* **Типы узлов**: Есть много типов узлов, например:
  * **HTTP Request** — для отправки запросов к веб-сервисам.
  * **Set** — для изменения или добавления данных.
*   **Соединение узлов**: Узлы соединяются между собой стрелками. Это позволяет передавать данные от одного узла к другому.

    Также можно провести стрелки от одного блока к нескольким и от нескольких блоков к одному.

Например, вы можете создать процесс, который:

* Получает данные из Google Sheets (узел).
* Обрабатывает их (другой узел).
* Отправляет результаты в Telegram (третий узел).

Таким образом, узлы в n8n помогают автоматизировать задачи и интегрировать разные сервисы без программирования
