# Триггеры в n8n

Триггеры в n8n – это специальные компоненты, которые запускают выполнение сценария (workflow) при наступлении определённых условий или событий. Они служат стартовой точкой для автоматизации процессов, позволяя системе реагировать на внешние события.

Аналогом треггиров n8n в Smartbot Pro являются блоки из [раздела “События”](https://docs.smartbotpro.ru/scenarii/sobytiya-scenariya).

## Примеры триггеров

1. **Webhook**: Этот триггер активируется, когда на него поступает HTTP-запрос из внешней системы. Например, вы можете настроить его для получения данных из формы на вашем сайте.
2. **Schedule**: Этот триггер позволяет запускать сценарий по расписанию, например, каждый день в определённое время.
3. **Интеграции с другими сервисами**: Триггеры могут отслеживать события в сторонних приложениях, таких как создание новой записи в Google Sheets или получение сообщения в Telegram.

{% note alert %}

Не подключайте к n8n Telegram бота, который уже подключен у вас в Smartbot Pro - он не будет корректно работать, лучше создайте отдельного бота или используйте Smartbot Pro для обработки сообщений в боте, а в n8n используете HTTP-запросы и [API Telegram](https://core.telegram.org/bots/api#available-methods))

{% endnote %}
