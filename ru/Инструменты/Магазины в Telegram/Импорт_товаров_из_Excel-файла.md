# Импорт товаров из Excel-файла

Импорт позволяет загрузить каталог с вложенными категориями и товарами со всеми необходимыми характеристиками. Вы можете как добавлять новые товары, так и обновлять существующие.

{% note info %}

Поддерживаемый формат файла для импорта -  Excel (xls, xlsx).

{% endnote %}

### Импорт товаров в новый магазин&#x20;

1. Перейдите в раздел "Магазины".
2. Нажмите кнопку "Создать магазин" и заполните раздел "Общие настройки", нажмите "Продолжить". \
   \
   Вы всегда сможете вернуться к общим настройками позже, если захотите изменить выбранные на этом шаге параметры.&#x20;
3. На странице "Загрузка товаров" выберите "Загрузить товары из файла".
4. Если вы используйте импорт товаров в Smartbot впервые, обязательно скачайте образец файла.&#x20;
5. Приведите файл со списком ваших товаров в соответствие образцу.&#x20;
6. Нажмите на кнопку "Загрузить файл" и выберите подготовленный файл со списком товаров.&#x20;

{% note info %}

**Обязательные поля:** Артикул, Категория 1, Название товара.

{% endnote %}

{% note info %}

**Артикул** - ключевой параметр товара. Он является обязательным. Указанный в файле Артикул будет однозначно определять каждый товар в базе данных Smartbot и позволит исключить их дублирование.

Обратите внимание, что **обновление существующих товаров** и создание новых будет происходить по полю "Артикул". Если товар с таким Артикулом уже был добавлен в магазин и загружается повторно с новыми характеристиками - товар будет обновлен.&#x20;

**Для создания нового товара** его Артикул не должен повторять Артикул существующих товаров.

{% endnote %}

### Импорт изображений

1. Загрузите изображение в одно из облачных хранилищ на выбор: Яндекс.Диск или Google Drive.

{% note info %}

Важно использовать именно эти хранилища - ссылки на изображения из других источников будут отклонены системой при загрузке.

{% endnote %}

2. Скопируйте ссылку на изображение: обычно для этого нужно кликнуть по картинке правой кнопкой мыши и выбрать в меню пункт "Скопировать ссылку" или "Поделиться" → "Скопировать ссылку". &#x20;

{% note info %}

Проверьте настройку общего доступа к конкретному изображению или папке, в которой располагается картинка. Для этого разрешите доступ по ссылке - доступа на просмотр будет достаточно.&#x20;

{% endnote %}

3. Вставьте ссылку в колонку "Изображения" в вашей таблице. Чтобы добавить несколько картинок, укажите несколько ссылок, разделяя их точкой с запятой (без пробела). \
   \
   Например, https://drive.google.com/file/d/image1/view?usp=share\_link;https://drive.google.com/file/d/image2/view?usp=share\_link



### Ошибки при импорте файла

Если в формате файла будут несоответствие с приведенным образцом, появится модальное окно со списком ошибок.
