# Пользователи


<p class="yfm-lead">Этот раздел позволяет отслеживать данные ваших пользователей в Smartbot Pro</p>
Войти в раздел можно из меню вашего проекта.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FVl7GvrVys9eiDL2givpQ%2Fimage.png?alt=media&#x26;token=96d86ea4-cb73-4319-80f2-f4af73bc2bdf" alt=""><figcaption></figcaption></figure>

{% note info %}

Здесь отображаются люди, которых можно сортировать по спискам, каналам и чатам.

{% endnote %}

В список "Все пользователи" по умолчанию добавляются все, кто взаимодействовал с вашим ботом вне зависимости от подписки на сообщество. Достаточно того, что человек написал вам.

Вы можете выбрать конкретный список людей для просмотра и канал (группу ВК/telegram-канал)

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2Fz0K4eI0bgpeWF6woEvQS%2Fimage.png?alt=media&#x26;token=65123d66-c9b8-4fe2-83f4-9afddc28f6a5" alt=""><figcaption></figcaption></figure>

Если нажать на имя пользователя, справа появится вспомогательное меню с детальной информацией о нем: какие у него значения переменных, в каких списках состоит, в какие чаты писал и какие отложенные события для него запланированы.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F2mL5eTspMEfjQf5JU9US%2Fimage.png?alt=media&#x26;token=d69b1125-539c-4a4f-8c02-176867f36929" alt=""><figcaption><p>Значения локальных переменных можно поменять в этом же окне</p></figcaption></figure>

{% note info %}

Обратите внимание: добавить в список Телеграм-бота можно только такого человека, который уже писал в вашего бота (или запускал его); для ВК-групп это ограничение не действует.

{% endnote %}

Кроме того, пользователей можно добавлять и выгружать таблицей Excel.&#x20;

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F4uOJ9QCCGgnFVMr62xkB%2Fimage.png?alt=media&#x26;token=1bfba221-9344-4738-a533-14348b30711b" alt=""><figcaption></figcaption></figure>

Если указать переменные через "Добавить переменную", тогда в разделе будут отображаться значения переменных у пользователей и можно выгрузить значения переменных в Excel таблице.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FrRVQKDzycEs1Pjhra9It%2Fimage.png?alt=media&#x26;token=a71e766c-d5d7-4894-a855-3cfd0c9aaee5" alt=""><figcaption></figcaption></figure>

\
Теперь вам известно, как работать с пользователями в Smartbot Pro!
