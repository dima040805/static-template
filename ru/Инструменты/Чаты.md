# Чаты

Раздел "Чаты" в Smartbot Pro - это все ваши переписки с пользователями, собранные в едином месте!

{% note info %}

Важно: раздел находится в бета-тесте, и на данный момент переписки в нем хранятся 1 месяц, после чего удаляются.

{% endnote %}

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FQz8uCnsirG1E5x4EvPo6%2Fimage.png?alt=media&#x26;token=4cbf21d2-cfb0-4bfc-a8e8-614eb1a3b338" alt=""><figcaption></figcaption></figure>

Здесь находятся все диалоги из ВКонтакте, Telegram.&#x20;

{% note info %}

Обратите внимание: в этот раздел попадают только переписки из ЛС (не комментарии и не беседы)

{% endnote %}

С помощью инструментов поиска вы можете выбрать, чаты с какими пользователями вывести. Сортировать можно по каналам и спискам.&#x20;

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F8KLgGRx1YKSuKtbt41Ti%2Fimage.png?alt=media&#x26;token=df81dde7-fc30-4dba-92d8-289ab0e3bbd4" alt=""><figcaption></figcaption></figure>

&#x20;

Когда вы лично отвечаете пользователю, бот автоматически прекращает выполнять сценарий, т.е. игнорирует все команды пользователя.&#x20;

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FkVealqFmZkWiR8DYntZE%2Fimage.png?alt=media&#x26;token=37e6ca4d-c188-464d-9dfd-d4bd8ca4381b" alt=""><figcaption></figcaption></figure>

Чтобы возобновить общение с ботом, после того, как вы закончили диалог, нужно запустить бота вручную одним из описанных способов:&#x20;

* с того места сценария, где он остановился
* с самого начала (бот выйдет из сценария и будет ждать команду)
* с конкретного сценария и блока (вы сможете выбрать вручную)

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FkoASFPF1FRG3y4IQKYHH%2Fimage.png?alt=media&#x26;token=f3537874-15b5-408d-81c4-d210fd219686" alt=""><figcaption></figcaption></figure>

Надеемся, что раздел "Чаты" окажется для вас удобен и сэкономит много времени, ведь для общения с клиентами больше не нужно переключаться на мессенджеры!&#x20;

### Лайфхак

Вы можете легко сформировать и передавать ссылку на чат с пользователем вашему менеджеру, чтобы он в один клик смог перейти к общению с клиентом.

Вот пример такой ссылки:

{% note info %}

h[ttps://testroom.smartbotpro.ru/project/61dbe38de16709acf09ad27b/conversations/%](https://testroom.smartbotpro.ru/project/61dbe38de16709acf09ad27b/conversations/6357b62f3baca072f544c6c1)user\_oid%

{% endnote %}

Чтобы автоматически создать и вставить в уведомление менеджеру ссылку на чат с нужным пользователем, выполним несколько простых шагов:

1. Заходим в раздел "Чаты"&#x20;
2. Открываем любой диалог
3. Копируем ссылку из адресной строки:

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FLOzt5656OIDpKhC8eUdh%2Fimage.png?alt=media&#x26;token=9bd19dc5-e146-4a51-adb9-5ead96f79022" alt=""><figcaption></figcaption></figure>

4. Последний шаг - просто заменяем в ней всё после `conversations/` на `%user_oid%` , где `%user_oid%` - переменная, значение которой при отправке в боте заполнится id (уникальный идентификатор) пользователя, в диалог с которым нужно перейти. Таким образом бот автоматически подставит id нужного пользователя. &#x20;



{% note info %}

Обратите внимание: чтобы ссылка отправилась корректно, нужно предварительно делинковать ее в блоке "Отправить уведомление". Для этого просто выделаем всю ссылку и нажимаем сюда:

{% endnote %}
