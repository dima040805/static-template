# Режим "Получить телефон"

Цель режима понятна из названия – получить заявку в виде телефона от пользователя.

{% note info %}

При использовании этого режима в настройках блока «Общение с AI ботом» вы увидите галочку "Выходить по достижении цели "Получить телефон"".

![](https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FrPXfO7VY2gunkrne1rWq%2Fimage.png?alt=media\&token=4fcd85f7-2d24-4556-848d-eb8f903d62be)

{% endnote %}

В отличие от режима "Проконсультировать", здесь можно настроить цель использования бота. Автоматически предлагается «Продажа товаров/услуг в интернете», но вы можете указать свое назначение для бота.&#x20;

Функциональное отличие также касается процесса обучения бота – вместо обкачивания сайта или документа, у вас есть 3 поля, которые можно заполнить для более точного формирования ответа бота. Поля названы в соответствии с информацией, которая поможет подробнее узнать о вашей деятельности - «Чем занимается компания?», «Часто задаваемые вопросы» и «Ограничения/подсказки для бота». Заполните поля, опираясь на пример, находящийся под каждым из полей.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FdqneTlY106NgCsYui1ki%2Fimage.png?alt=media&#x26;token=8122ca32-9572-4a8f-8fd0-d14ecbb1ab6b" alt=""><figcaption><p>Пример заполнения для поля "Чем занимается компания?"</p></figcaption></figure>

{% note info %}

На все поля стоит общее ограничение 5500 символов. Т.е. если в каждом из полей будет по 3000 символов (суммарно 9000), то бот подскажет, что 3500 символов будут лишними.

{% endnote %}

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FBcv8mDWgf4vIYea7nfPH%2Fimage.png?alt=media&#x26;token=fa9d930a-958c-48d6-8eac-012348d3457c" alt=""><figcaption></figcaption></figure>

После того, как вы заполнили все необходимые поля,  нажмите кнопку "Сохранить", чтобы применить все настройки и теперь вы можете проверить ответы AI-бота в тестовом окне справа (у вас есть 100 бесплатных тестовых сообщений).
