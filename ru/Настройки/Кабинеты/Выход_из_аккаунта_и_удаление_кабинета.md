# Выход из аккаунта и удаление кабинета

Для того, чтобы выйти из аккаунта, нажмите на профиль в правом верхнем углу

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FBcowLIySC7akiCnMQpB5%2Fimage.png?alt=media&#x26;token=fe4000d9-2425-46b1-9459-560c8932c3c3" alt=""><figcaption></figcaption></figure>

Для удаления кабинета и данных о всех проектах перейдите в настройки кабинета. Для этого нажмите на иконку шестеренки рядом с нужным кабинетом.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FZL96OL1hhOW6a0p4Hbqb%2Fimage.png?alt=media&#x26;token=367ee7c2-8bd2-4ec2-9b7d-65a75834a93a" alt=""><figcaption></figcaption></figure>

На открывшейся странице перейдите в настройки и нажмите кнопку "Удалить".&#x20;

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2Fv3uWSirAt4luJntWVjjA%2Fimage.png?alt=media&#x26;token=6ad1a06c-4ebb-42f7-b66f-53031d935269" alt=""><figcaption></figcaption></figure>

{% note info %}

Удаление кабинета удалить все проекты и все ваши данные: подключенные боты, списки пользователей, сценарии, статистика и так далее.  Это действие невозможно будет отменить!&#x20;

{% endnote %}
