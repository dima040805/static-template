# Как работает оплата AI-ботов

### Пробный период и подписка

После регистрации вы получаете 7 дней бесплатного доступа. По завершении пробного периода потребуется оформить подписку — для этого нужно пополнить баланс. Стоимость подписки и текущий баланс отображаются в настройках кабинета.

📌 Подробнее о тарифах и подписке — в [отдельной статье](https://docs.smartbotpro.ru/dokumenty/podpiska-i-tarify).

### Откуда списываются средства?

Средства за ответы AI-ботов списываются с того же баланса, который используется для оплаты подписки.

{% note warning %}

Если вы активно используете AI-ботов, имейте в виду: баланс может закончиться раньше, чем истечёт срок подписки.

{% endnote %}

### Как пополнить баланс

Пополнить баланс можно в двух местах:

* в настройках кабинета
* на странице баланса внутри AI-бота

### Сколько стоит ответ AI-бота?

Стоимость зависит от выбранной модели:

| Модель           | Стоимость ответа |
| ---------------- | ---------------- |
| ChatGPT-4o       | 3 ₽              |
| ChatGPT-4o mini  | 0.5 ₽            |
| YandexGPT Pro    | 3.5 ₽            |
| ChatGPT-4.1 nano | Бесплатно        |
| ChatGPT-4.1      | 1.5 ₽            |
| ChatGPT-4.1 mini | 0.5 ₽            |
| ChatGPT o1       | 15 ₽             |

{% note warning %}

Важно: списание происходит только при успешном ответе. Если бот не смог ответить (например, из-за ошибки или отсутствия данных), средства не списываются.

{% endnote %}

### Как посмотреть траты на AI-ботов?

1. Откройте нужного бота в разделе Smartbot AI
2. Перейдите во вкладку "Баланс"
3. Ниже будет указана сумма списаний за ответы AI.

⚠️ Обратите внимание: учитываются все ответы AI-ботов в рамках одного проекта.\
Если у вас несколько проектов, просмотрите статистику отдельно для каждого.

### Можно ли протестировать AI в пробный период?

Да! После настройки бота (добавления контекста, инструкций и обучающих материалов) вы можете протестировать его в виджете справа.

Каждому новому AI-боту в тестовом режиме доступно 100 бесплатных сообщений. При каждом пополнении баланса этот лимит обновляется для всех AI-ботов.
