# Как активировать подписку на сервис

Есть несколько вариантов активации подписки. Вы можете:&#x20;

1. Оплатить месячную стоимость сервиса по текущему тарифу&#x20;
2. Оплатить несколько месяцев использования сервиса по вашему текущему тарифу и получить скидку
3. Оплатить произвольную сумму, но не менее стоимости минимального тарифа

Вне зависимости от способа активации, внесенная сумма будет зачислена на баланс кабинета. Оттуда ежедневно в 12:00 по Мск списывается сумма по текущему тарифу, который зависит о размера собранной базы пользователей по всем подключенным каналам.&#x20;

{% note info %}

**Важно!**

Стоимость подписки рассчитывается по количеству контактов пользователей в базе Smartbot (страница "Тариф и баланс" в настройках кабинета).

Например, в вашей группе ВК 10 000 подписчиков, но в базе Smartbot всего 100 (то есть только у 100 из них был контакт с ботом), то тариф будет рассчитываться для 100 человек.

Пользователи попадают в базу контактов после переписки с ботом или если импортировать базу вручную через раздел "Пользователи".&#x20;

{% endnote %}

Если количество пользователей увеличится, кабинет перейдет на другой тариф в соответствии с новым числом пользователей и ежедневная сумма списания также изменится.&#x20;

{% note info %}

Не беспокойтесь, мы заранее направим предупреждение о предстоящем списании на электронную почту создателя и администраторов кабинета.&#x20;

{% endnote %}

Также системное уведомление отправит один из созданных ботов тем администраторам, которые привязали свой канал в социальной сети.&#x20;

Вы также будете получать уведомления:&#x20;

1. Об изменениях тарифа и его стоимости
2. О завершении пробного периода
3. О результате автоматического списания
4. О недостатке средств на балансе

{% note info %}

Чтобы ваш бот работал бесперебойно и не выключился в неподходящий момент, **первая попытка списания происходит за 3 дня до окончания баланса кабинета.**&#x20;

Если списание не удалось, следующая попытка происходит на следующий день, и еще одна попытка в последний день. &#x20;

{% endnote %}
