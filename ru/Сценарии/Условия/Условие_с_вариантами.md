# Условие с вариантами

{% note info %}

Условия позволяют разветвлять сценарий, направляя его, в зависимости от полученных сообщений и переменных.

{% endnote %}

<p class="yfm-lead">Рассмотрим два основных типа условий в Smartbot Pro</p>
Например, если пользователь пришел в ВК, для него можно запустить одну цепочку, а для тех, кто писал в telegram - уже другую.

Чтобы добавить разветвление, создадим на экране блок простого условия. Сделать это можно, щелкнув правой кнопкой мыши в свободном месте экрана, либо через левое вспомогательное меню.

### Особенность условий с вариантами

Условие с вариантами позволяет прописать сразу много действий для разных значений одной и той же переменной. К примеру, можно всего одним блоком сделать разные ответы на разные приветствия.&#x20;

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FU5c9DZpWQYT3nTF8f8P3%2Fimage.png?alt=media&#x26;token=09a5a482-771b-41e0-983a-5bce69b861de" alt=""><figcaption></figcaption></figure>

Или сделать многоуровневую проверку счетчика баллов.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FQ3Eq6tdV4p9eMM5eu0Vn%2Fimage.png?alt=media&#x26;token=392a7b33-a21e-451f-bbfa-f69d1e68d00d" alt=""><figcaption><p>Таким образом, человек с 6 баллами получит одно сообщение, а тот, у кого их, например, 17 - другое</p></figcaption></figure>

{% note info %}

Условие с вариантами сэкономит вам пространство на экране сценария, если нужно сравнить одну переменную (или сообщение) с несколькими значениями &#x20;

{% endnote %}
