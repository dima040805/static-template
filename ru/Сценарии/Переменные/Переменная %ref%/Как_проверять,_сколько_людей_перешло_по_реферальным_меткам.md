# Как проверять, сколько людей перешло по реферальным меткам

Один из способов узнать, сколько человек перешло по реферальной ссылке - зайти в раздел «Пользователи».

Для просмотра тех, кто перешел по ссылке с %ref%, в разделе «Пользователи» необходимо выбрать нужный канал и нажать «Добавить переменную»

Далее: Нажать галочку «Источник перехода(%ref%)» и «Выбрать»

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdZXZtqGhr_cLhooKlG5_sUU1fx-ZIgq8P3dJ1WNHFNIVkHU_Xfv8crpdjTq08JTVE1lOTPsZbdzOtbppkFBZpmYb1j3qb8fD2dcdYqHWypKJ81LV244SNfQlj_rAEDZ0mTICZI?key=90t4swd-sTso0Ca3Y1EGqLjn" alt=""><figcaption></figcaption></figure>

{% note tip %}

Теперь в списке пользователей отображаются их реферальные метки. Вы можете скачать список в виде и отследить, сколько людей перешло по каждой из них.&#x20;

{% endnote %}

### Настройка отображения на графике

Если хотите отслеживать все переходы по ссылкам с параметром %ref% в разделе “Статистика”, то в сценарий нужно добавить блок “Записать в статистику”:

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdrAkB54gHGXozqZGp57FnBlJBHkaPk7lmd_zZibRR1SFlO2HXKA1Wp4AChVdQqDCxUFkoPJLytgHxEooPEGSg4CsLvmsp-VKiORi37GidJMLLytmDHKyUubTfhoSzH8o5h7bI7nQ?key=90t4swd-sTso0Ca3Y1EGqLjn" alt=""><figcaption></figcaption></figure>

Это можно сделать после блока, где проверяется условие на переменную %ref%.&#x20;

Далее нужно создать или выбрать нужную метку:

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdzzH6wU6soAWEKwiP3NI7ZlrX9h982_DJ4I6723ituRHSKM_uinAF54IbG3C5M4yOtcvX6r6aaEPP0FC1bo4ZOV75YgDmvu6wkimzhljG7fEmBcTUKxnhXTOxBSplZ7yg2mbZd?key=90t4swd-sTso0Ca3Y1EGqLjn" alt=""><figcaption></figcaption></figure>

Дать ей название и нажать “Создать”

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXcQPoWYj27p-nTtE8DfUkGJbKIXvLiZrxOEahrKgRJKJ04DlGPqh21-ftMO50Y1yRjCCpcQx3Xug7LaxcQPkt8UPZmipAuUsudmbJXh7s3otY5mkhuS1yfgz11Pp-5i55Krbqv0?key=90t4swd-sTso0Ca3Y1EGqLjn" alt=""><figcaption></figcaption></figure>

Теперь когда наш пользователь перейдет с параметром “market”, попадет в статистику с названием “Переход по реферальной ссылке - market”

Чтобы посмотреть кто перешел по этой ссылке - переходим в раздел “Статистика” и выбираем событие, в котором сохранен переход по ссылке с параметром %ref%, и проставляем нужный период:&#x20;

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdoWyeerPTrXCsJni2RLSJyOKVhnLNRQ4MPiUfglsJxwiaDpanxQ_Twq-n86aYnP49ZareUGzYb3tbf1-3nZ9EAIZljR-TqUC0IYg1i58vkAzxsPVPaPRjlhRWzObOIfpr3dk7Q8Q?key=90t4swd-sTso0Ca3Y1EGqLjn" alt=""><figcaption></figcaption></figure>

Если у вас много меток, и нужно отслеживать каждую, то имеет смысл использовать блок "Условие с вариантами"&#x20;
