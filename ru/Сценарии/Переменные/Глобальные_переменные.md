# Глобальные переменные



{% note info %}

**Глобальные переменные** хранят одинаковое значение для всех пользователей

{% endnote %}

Например, вы создали глобальную  переменную %количество\_заявок% и хотите сохранять в ней количество пользователей, которые оставили заявку на ваш товар.&#x20;

В таком случае, в конце сценария по сбору заявок вы можете добавить шаг "установить переменную", который будет прибавлять единицу к переменной %количество\_заявок%.&#x20;

Так вы сможете хранить именно общее количество заявок: каждая новая будет учтена. Бот будет увеличивать одну и ту же переменную независимо от того, с каким пользователем он  ведет диалог. Для подобных целей отлично подходит глобальная переменная.

А с помощью глобальной переменой-массива можно раздавать лимитированные промокоды.&#x20;
