# Работа с пользователем

{% note info %}

С помощью этого действия вы сможете добавлять и удалять пользователей из списков. Списки можно использовать в рассылках, выгрузках и условиях сценариев.&#x20;

{% endnote %}

<p class="yfm-lead">Как формировать списки пользователей</p>
Действие "Работа с пользователем" включает в себя два шага:&#x20;

1. Добавить пользователя в список&#x20;
2. Удалить пользователя из списка&#x20;

Выберем "Добавить в список" в дополнительном меню слева или, щелкнув  правой кнопкой мыши в свободном месте экрана.&#x20;

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2Fj5NVZfMbZtGFBEhTIT3M%2Fimage.png?alt=media&#x26;token=e1931850-abc3-49fe-9a21-5f6f403f9366" alt=""><figcaption></figcaption></figure>

В меню блока (справа) можно выбрать существующий список,&#x20;

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FRZrewNvzRkM8R9AuvaWV%2Fimage.png?alt=media&#x26;token=7d5f357c-2213-4788-b338-74ec8bc2a625" alt=""><figcaption></figcaption></figure>

или создать новый

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FmnAdgfiWP5ZsZhHm72Uk%2Fimage.png?alt=media&#x26;token=78f75bb4-275a-4101-97fc-e73aefd816e3" alt=""><figcaption></figcaption></figure>

Список со всеми его участниками доступен в разделе "Пользователи".&#x20;

Там же есть отдельный подраздел, в котором можно редактировать и удалять списки вручную.

Так в Smartbot Pro можно собирать пользователей в списки по интересам. С помощью данного действия вы можете дать человеку возможность подписаться на интересующую рассылку (или, напротив, отписаться).
