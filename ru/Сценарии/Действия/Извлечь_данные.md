# Извлечь данные

{% note info %}

Действие "извлечь данные" - незаменимый помощник в получении и сохранении информации. С помощью него можно извлечь из любой строки любое содержимое в заданном формате.

{% endnote %}

Добавим блок, выбрав действие в дополнительном меню слева или щелкнув правой кнопкой мыши на свободном участке экрана.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F2WTU8ddvLckwxD8LCgkU%2Fimage.png?alt=media&#x26;token=28e35797-e888-4b70-a807-8db8644043da" alt=""><figcaption></figcaption></figure>

Меню блока действия содержит **входное значение, шаблон сообщения**, **фильтр** и **переменную**, в которую нужно записать данные.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F3gz6QyZSE6K2pF2dkxcE%2Fimage.png?alt=media&#x26;token=c19a0391-5924-480b-b48a-9be30068500c" alt=""><figcaption></figcaption></figure>

Поле "Входное значение" - это указание любой строки, из которой нужно произвести извлечение.&#x20;

{% note info %}

Если при выполнении действия указать входным значением переменную %message\_text%, то бот возьмет данные из последнего сообщения подписчика

{% endnote %}

Начните вводить шаблон, и появятся фильтры.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F3ABJ0n4JE4ZZfaFsumdn%2Fimage.png?alt=media&#x26;token=9ae20d5d-0ea4-4757-965b-7ff1b54377d7" alt=""><figcaption></figcaption></figure>

Вы можете указать один из фильтров и его настройки, а затем установить переменную, в которую записать полученные данные.

Кроме этого, вы можете задать **любой формат** строки и извлечь нужные данные из ее частей с помощью шаблона.
