# Изменение сделки

Прежде чем изменять сделку, нужно ее для начала создать. Если вы ещё не знакомы с созданием сделки, сначала ознакомьтесь [с этой инструкцией](https://www.notion.so/279cf4468eb18040ab53d4d793aa5012?pvs=21).

### **SQ: подготовка ответа**

**Шаг 1:** Создайте блок **“Выполнить SmartQuery”** и напишите следующий код:

```jsx
%response% = {}

%response_status% = 0

%req_body% = {
    "FIELDS": {
        "NAME": "Иван",
        "LAST_NAME": "Иванов"
    },
    "ID": %deal_id%
}

```

Более подробно с настройкой изменения сделки вы можете ознакомиться в [официальной документации Bitrix24](https://apidocs.bitrix24.ru/api-reference/crm/deals/crm-deal-update.html). Используйте соответствующие поля для изменения нужных вам данных.

### **HTTP: изменение сделки**

**Шаг 2:** Создайте блок **“Отправить HTTP-запрос”.**

Оставьте метод `POST` без изменений.

В поле **“URL”** введите:

```jsx
%BITRIX_CRM_WEBHOOK%/crm.deal.update 
```

Не обращайте внимание на уведомление “Некорректная ссылка”.

Выберите в **“Сохранить ответ в”** переменную `%response%` (или ваше название переменной для хранения ответа — это должен быть словарь).

Выберите в **“Сохранить статус ответа в”** переменную `%response_status%` (или ваше название переменной для хранения статуса ответа — это должно быть число).

В **“Тело запроса”** выберите **“Текст”** и введите:

```json
{{ %req_body% | to_json }}
```

![](https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2F52c98b2f-cfa6-481c-a1d3-54460290a07d%2Fimage.png/size/w=2000?exp=1761904996\&sig=k5Wphj7KlkzhBYN7H2bi8DtoX8K76WDFCaaZS_5skm8\&id=27dcf446-8eb1-80ba-9b1e-df5f5a635ab9\&table=block\&userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1)

Добавьте заголовки:

`content-type` = `application/json`

`accept` = `application/json`&#x20;
