# Создание сделки

{% note info %}

Из этой статьи вы узнаете, как создать сделку в **Bitrix24** с помощью [официальной документации CRM.](https://apidocs.bitrix24.ru/api-reference/crm/deals/crm-deal-add.html)

{% endnote %}

### **SQ: подготовка ответа**

**Шаг 1:** Создайте блок **“Выполнить SmartQuery”** с кодом:

```json
%response% = {}

%response_status% = 0

%req_body% = {
    "FIELDS": {
        "TITLE": "Тестовая сделка",
        "CONTACT_IDS": [
            %contact_id%
        ],
        "UTM_SOURCE": "tg_bot"
    }
}
```

При создании сделки вы можете указать множество полей – `FIELDS`, список и описание которых вы можно изучить в документации [Bitrix24](https://apidocs.bitrix24.ru/api-reference/crm/deals/crm-deal-add.html).

Для нашего примера заполняются следующие поля:

* `TITLE` — название сделки;
* `CONTACT_IDS` — массив ID контактов, которые будут прикреплены к сделке (Более подробно о создании ID контакта вы можете узнать [здесь](https://www.notion.so/279cf4468eb1802cb6ace4e422e3eada?pvs=21));
* `UTM_SOURCE` — источник, откуда пришел контакт.

### **HTTP: с**оздание сделки

**Шаг 2:** Создайте блок **“Отправить HTTP-запрос”** с **URL**:

```jsx
%BITRIX_CRM_WEBHOOK%/crm.deal.add
```

Оставьте метод `POST`без изменений. Выберите в **“Сохранить ответ в”** переменную `%response%` (или ваше название переменной для хранения ответа — \*\*\*\*это должен быть словарь). Выберите в **“Сохранить статус ответа в”** переменную `%response_status%` (или ваше название переменной для хранения статуса ответа — это должно быть число). В **“Тело запроса”** выберите **“Текст”** и введите:

```json
{{ %req_body% | to_json }}
```

`content-type` = `application/json`

`accept` = `application/json`

### **SQ: ID сделки**

**Шаг 3:** Создайте блок **“Выполнить SmartQuery”** с кодом:

```json
%deal_id% = %response%['result']
```

`%deal_id%` должна быть пользовательской переменной типа **“Число”**. Создайте ее, название переменной может быть любое.

В этой переменной будет храниться **ID сделки с этим пользователем в CRM**.

Готово! Вы настроили создание сделки в **Bitrix24** и сохранение ID созданной сделки в пользовательскую переменную Smartbot.
