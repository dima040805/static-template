# Создание компании

{% note info %}

Из этой статьи вы узнаете, как создать компанию в Bitrix24 с помощью [официальной документации CRM](https://apidocs.bitrix24.ru/api-reference/crm/companies/crm-company-add.html).

{% endnote %}

### **SQ: подготовка данных**

**Шаг 1:** Создайте блок **“Выполнить SmartQuery”** с кодом:

```json
%response% = {}

%response_status% = 0

%req_body% = {
    "FIELDS": {
        "TITLE": "Ромашка",
        "ADDRESS": "ул. Ромашковая, д.7, стр.1"
    }
}
```

При создании компании вы можете указать множество полей — `FIELDS`, список и описание которых мы получим ниже.

Для примера заполняются следующие поля:

* `TITLE` — название компании (**обязательное поле**, `IsRequired = true`);
* `ADDRESS` — адрес компании.

### **Как узнать доступные поля компании**

Перед созданием компании важно определить, какие поля можно и нужно указывать. Для этого:

1. Откройте [**Postman**](https://www.postman.com/).
2.  Нажмите “**Create new request”**.



    <figure><img src="https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2Fd965e1b7-8b0f-406a-be15-153a46735a17%2Fimage.png/size/w=2000?exp=1760693086&#x26;sig=Cp9v1cFLe6YnUfJ7ixum7xPN8LwJTpWX4AOJI-D-pz8&#x26;id=28dcf446-8eb1-8037-8bd5-ced0b4965858&#x26;table=block&#x26;userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1" alt=""><figcaption></figcaption></figure>
3. Установите метод `POST`.
4.  Вставьте ваш **URL** вебхука:

    ```jsx
    https://<Ваш_вебхук>/crm.company.fields
    ```



    <figure><img src="https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2Fb9ace69c-2214-4a00-8333-4de1f278a703%2Fimage.png/size/w=2000?exp=1760692924&#x26;sig=X21difUcIpH6FwLAdrx1ltSg8m9xulIHXmdjxZF3uVI&#x26;id=28dcf446-8eb1-80f8-b16c-db8f61bc92fd&#x26;table=block&#x26;userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1" alt=""><figcaption></figcaption></figure>
5. Отправьте запрос кнопкой “**Send”**.

<figure><img src="https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2F32266c28-e569-4607-944d-d4620fc795b9%2Fimage.png/size/w=2000?exp=1761203587&#x26;sig=FFaqAUuFF_XQyVVLGAGGJe7E6dtRsOGicIprn_rDLPA&#x26;id=293cf446-8eb1-8031-889b-fd9ed2dbc2a6&#x26;table=block&#x26;userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1" alt=""><figcaption></figcaption></figure>

Пример фрагмента результата:

```json
{
    "result": {
        "ID": {
            "type": "integer",
            "isRequired": false,
            "isReadOnly": true,
            "isImmutable": false,
            "isMultiple": false,
            "isDynamic": false,
            "title": "ID"
        },
        "TITLE": {
            "type": "string",
            "isRequired": true,
            "isReadOnly": false,
            "isImmutable": false,
            "isMultiple": false,
            "isDynamic": false,
            "title": "Название компании"
        }
    }
}
```

Ключ `type` указывает **тип данных**, который это поле принимает.

<figure><img src="https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2Fa829730c-240b-48d3-9101-112512421b69%2Fimage.png/size/w=2000?exp=1761203630&#x26;sig=DF64BfmawOygoz6AKL2FiSO5JaPf1nXZ3n-FmYnlLE4&#x26;id=28dcf446-8eb1-801a-addb-f75a4607b64c&#x26;table=block&#x26;userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1" alt=""><figcaption></figcaption></figure>

Поле `IsRequired = true` означает, что оно **обязательно к заполнению** при создании компании.

<figure><img src="https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2F795ba69b-426c-4830-9513-1b7fd6f5d033%2Fimage.png/size/w=2000?exp=1761203648&#x26;sig=h_WoDX_0eOPzC4M7xNcmtIGkE2G2w6R3G08IhefVuu4&#x26;id=28dcf446-8eb1-8048-8b3c-dffe3ad21522&#x26;table=block&#x26;userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1" alt=""><figcaption></figcaption></figure>

### **HTTP: создание компании**

**Шаг 2:** Создайте блок **“Отправить HTTP-запрос”** с **URL**:

```jsx
%BITRIX_CRM_WEBHOOK%/crm.company.add
```

<figure><img src="https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2Fa4bd9a49-0384-4008-9755-176d51f4942d%2Fimage.png/size/w=2000?exp=1760693226&#x26;sig=37TbEvOoXmEvfNKJElasuQFmJ8xz22cR_hdHayetg8Y&#x26;id=28dcf446-8eb1-805d-a02c-f05365193d9c&#x26;table=block&#x26;userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1" alt=""><figcaption></figcaption></figure>

Оставьте метод `POST`без изменений. Выберите в **“Сохранить ответ в”** переменную `%response%` (или ваше название переменной для хранения ответа — \*\*\*\*это должен быть словарь). Выберите в **“Сохранить статус ответа в”** переменную `%response_status%` (или ваше название переменной для хранения статуса ответа — это должно быть число). В **“Тело запроса”** выберите **“Текст”** и введите:

```json
{{ %req_body% | to_json }}
```

Добавьте заголовки:

`content-type` = `application/json`

`accept` = `application/json`&#x20;

### **Тестирование**

1. Наведите курсор на три точки рядом с блоком “**SmartQuery”.**<br>

<figure><img src="https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2Fefe00e00-f8f7-4523-98e4-0b74466b73d1%2Fimage.png/size/w=2000?exp=1761203666&#x26;sig=vtTkOCnmIUJ8JdvoubwllOxrpMn3u_Yd1iuFa3T8EiA&#x26;id=28dcf446-8eb1-80a4-b2a3-e77733f89c61&#x26;table=block&#x26;userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1" alt=""><figcaption></figcaption></figure>

2. В открывшемся меню выберите “**Тестировать”** → “**Запустить”**.

<figure><img src="https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2F2f8e0e99-3efa-4183-8084-6a0058fb9e50%2Fimage.png/size/w=2000?exp=1761203692&#x26;sig=Pvvjc_QzFbwO6sKyyuC7BcCqC4Nbcfb9dAvlN6wpXL4&#x26;id=28dcf446-8eb1-8013-a909-c24b7eefe350&#x26;table=block&#x26;userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1" alt=""><figcaption></figcaption></figure>

3. Перейдите в **Bitrix24 → СRM → Клиенты → Компании.**

<figure><img src="https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2F77066a05-d2f8-4722-ae3d-16d5a43b4348%2Fimage.png/size/w=2000?exp=1761203713&#x26;sig=9pYPqcKUKyCBM_yu69lRrP4togiv1fxgLg8qtonDtFw&#x26;id=28dcf446-8eb1-8038-9103-d20788df1b50&#x26;table=block&#x26;userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1" alt=""><figcaption></figcaption></figure>

Во вкладке “**Компании”** появится созданная компания:

<figure><img src="https://img.notionusercontent.com/s3/prod-files-secure%2F233fe1fe-7932-4fbf-bb40-67d177e20dc4%2F33d423d0-87e2-45c6-a515-65c77751436a%2Fimage.png/size/w=2000?exp=1761203726&#x26;sig=ZXFbBxf2MBlWbYpUVaNJ5edIzxhNv5HwQicaBAPrP3o&#x26;id=28dcf446-8eb1-80c9-8ee6-e69e28dddbc3&#x26;table=block&#x26;userId=611a2099-ca7e-45f7-820f-80aaea4f6cd1" alt=""><figcaption></figcaption></figure>

**Готово!** Вы успешно настроили создание компании в **Bitrix24** через Smartbot.
