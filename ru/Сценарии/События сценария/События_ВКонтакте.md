# События ВКонтакте

Список событий ВК находится в левом меню, во вкладке "События"

### Лайк/удаление лайка и репост

> Событие работает только для групп ВК

С помощью этого события можно начислять баллы за лайки и репосты, а также предотвращать накрутку через удаление лайков.

Если публикация, как большинство постов ВК, состоит из текста и картинки, реакция на лайк сработает дважды (на пост и на картинку).

{% note info %}

Чтобы ваш бот среагировал на конкретный пост, откройте блок и выберите "Условие на ID объекта". Здесь же можно выбрать тип объекта: пост, картинка, видеозапись и т.д.

{% endnote %}

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F4O1oBryB3iMJ7ozFSdfP%2Fimage.png?alt=media&#x26;token=57ead3b3-a009-4771-a5e0-b1b64157a777" alt=""><figcaption></figcaption></figure>

Аналогичным образом работает удаление лайка и репост.

### Вступление в группу/выход из группы

> Доступно только для групп ВК

&#x20;Вы можете настроить действие, которое бот выполнит, если человек вступил в группу или подписался на сообщество, к примеру: начислить ему бонус (изменив переменную), записать событие в статистику или отправить сообщение.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FxuKzw2pf4W1A0l10Kn30%2Fimage.png?alt=media&#x26;token=59492603-0b76-41dd-afae-d4354cb72317" alt=""><figcaption></figcaption></figure>

{% note info %}

Обратите внимание: сообщение будет отправлено в ЛС пользователю, только если он разрешил группе писать ему в ЛС.

{% endnote %}

Если пользователь отписался, бот может отправить ему «прощальное письмо» или опрос, чтобы понять причины отписки.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FCRuONtKH7Kez3313iokL%2Fimage.png?alt=media&#x26;token=5b29e835-fec6-43f9-9507-f11fbdacd733" alt=""><figcaption></figcaption></figure>

Таким несложным образом, вы можете применять события ВКонтакте!&#x20;
