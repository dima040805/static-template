# События сценария

{% note info %}

**Событие** – внешнее воздействие, на которое может отреагировать бот. Например, поступление сообщения от подписчика.

{% endnote %}

Бот всегда запускает сценарий по какому-то из событий. Причем выбирает именно то событие, которое подходит лучше всего. Подробнее об этом можно прочитать в [этой статье](../Основы/Как_выбирается_сценарий.md).
