# Подключение платежной системы

Smartbot Pro поддерживает следующие платежные системы:&#x20;

* Qiwi
* ЮКassa
* ЮMoney
* Telegram Payments
* Prodamus
* Т-Банк
* Cloud Payments

Чтобы получать деньги от клиентов, сначала привяжем нужные платежные системы через раздел "Интеграции".

Когда ваш кошелек привязан к боту, остается только настроить выдачу платежных кнопок для пользователя.&#x20;

В любом сценарии создаем блок "Отправить сообщение" и добавляем кнопку.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FEo2Y7QNzAUbQCVg8y3Hz%2Fimage.png?alt=media&#x26;token=fe1ae063-bedf-4c43-84ea-16d390e5b353" alt=""><figcaption></figcaption></figure>

Нажимаем на иконку карандаша, чтобы отредактировать кнопку.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FYmJSVXrjhqaBpt1fI519%2Fimage.png?alt=media&#x26;token=5b3714a2-02cc-43c3-8e9a-1651441e9b21" alt="" width="304"><figcaption></figcaption></figure>

Выбираем тип кнопки "Создать платеж".

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FMzRebfKTI3Cxy9KvAzko%2Fimage.png?alt=media&#x26;token=5fad66ea-5795-4ac5-98d1-c303280afa32" alt="" width="309"><figcaption></figcaption></figure>

Обратите внимание, что для кнопки "Создать платеж" в каналах Telegram всегда должна быть включена настройка "Показывать кнопки внутри сообщения (inline).

{% endnote %}
