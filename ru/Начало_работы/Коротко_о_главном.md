# Коротко о главном

**Smartbot Pro** – это набор маркетинговых инструментов для мессенджеров. Smartbot Pro основан на сервисе [Smartbot](https://smartbot-vk.ru) для ВКонтакте, которым пользуются десятки тысяч групп ВК.

Задачи, которые решает смартбот:

1. Автоматизация продаж через мессенджеры
2. Оптимизация ресурсов на поддержку пользователей в соцсетях
3. Создание воронок продаж с разветвленной логикой, отложенными событиями, сбором статистики и различными инструментами повышения конверсий.

Сердцем сервиса являются **сценарии** бота

{% note info %}

**Сценарий** – последовательность шагов, которые выполняет бот в ответ на события, например, входящее сообщение в мессенджере, вступление в группу в соцсети или оплата товара в вашем онлайн-магазине.

{% endnote %}

Сценарий может быть привязан к нескольким **каналам**

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F3lk3MWD7DdWzAlWLU2sI%2Fimage.png?alt=media&#x26;token=bd0b8ace-7f95-43d7-9f06-b5a4842fc59d" alt=""><figcaption></figcaption></figure>

{% note info %}

**Канал** – среда общения с вашими подписчиками. Например, группа ВКонтакте или бот в телеграме.

{% endnote %}

Чтобы запустить бота с помощью Smartbot нужно создать сценарий ведения диалога и подключить нужный канал. Все это занимает совсем немного времени :sunglasses:

Один сценарий может быть подключен также к нескольким каналам разных типов: Telegram, Вконтакте, Viber и т.д.
