# Подключение Wazzup


<p class="yfm-lead">В этой статье мы разберем, как подключить Wazzup к Smartbot Pro.</p>
Подключение Wazzup зависит от того, есть ли в вашем аккаунте на данный момент интеграция с какой-либо CRM-системой, т.к. только внутри уже подключенной интеграции будет ключ для подключения к Smartbot Pro. Ниже рассмотрим каждый из вариантов.

### Если интеграция с CRM уже есть

1. Зайдите в кабинете Wazzup в раздел "Интеграция с CRM" и перейдите в подраздел "Дополнительно"

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FoBrtvjVuU81yCGswu8eU%2Fimage.png?alt=media&#x26;token=1e636419-6b84-4472-a127-da0008425903" alt="" width="563"><figcaption></figcaption></figure>

2. Внутри найдите поле "Ключ API" и скопируйте его значение

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FIMjjvDq7JNPDqZDe3eJh%2Fimage.png?alt=media&#x26;token=ee9ea53d-7bc9-4b31-b551-6e9c968d379a" alt="" width="563"><figcaption></figcaption></figure>

3. Перейдите в Smartbot Pro в раздел "Каналы" и выберите Wazzup

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FVfY2tKChmp5HqL68YH9T%2Fimage.png?alt=media&#x26;token=433172d7-cea7-4494-a58d-45cbfacecbeb" alt=""><figcaption></figcaption></figure>

4. Выберите вариант подключения с уже имеющейся интеграцией с CRM и введите в поле полученный ранее ключ

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FZQOWJQcokiPO2RL2qnJA%2Fimage.png?alt=media&#x26;token=19782494-f911-42ea-b2c3-40e9962ef2eb" alt="" width="563"><figcaption></figcaption></figure>

{% note tip %}

Аккаунт Wazzup с подключенной CRM успешно подключен к Smartbot Pro!

{% endnote %}

### Если интеграции с CRM пока что нет

1. Зайдите в кабинете Wazzup в раздел "Интеграция с CRM" и выберите "Подключение по API" (иконку шестеренки).&#x20;

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F3IqdmbOy218Sm6fv80Ni%2Fimage.png?alt=media&#x26;token=278c0bee-8c11-4ca7-9d9e-6dada59cdfa3" alt="" width="563"><figcaption></figcaption></figure>

2. В появившемся окне нажмите "Подключить"

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FxrRBBSduBf2HwIAtDpRn%2Fimage.png?alt=media&#x26;token=68054c9a-270b-47b7-8457-966fa183c9dd" alt="" width="486"><figcaption></figcaption></figure>

3. Тут и появится нужный нам ключ, скопируйте его

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FhszKUYTHtxK5BMk2TTDw%2Fimage.png?alt=media&#x26;token=72f3d5b3-5d69-4abc-9ed8-bded3415d0d0" alt="" width="480"><figcaption></figcaption></figure>

{% note info %}

Если внутри раздела нет иконок для выбора CRM, то скорее всего у вас уже подключена одна из доступных интеграций. Чтобы это проверить, в разделе "Интеграция с CRM" посмотрите на название страницы (будет справа от id вашего кабинета wazzup)

* если написано "Интеграция с \*название crm-системы\*", тогда вам нужно вернуться в начало статьи и найти главу "Если интеграция с CRM уже есть".![](https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2Fq2dlMKB7ycxslBsSJtSG%2Fimage.png?alt=media\&token=f382612a-54d4-4119-bf45-112e900c13fc)
* если написано "Интеграция с API" (см скрин ниже)\
  ![](https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F5i2MnEeltxj2LkwE7MFZ%2Fimage.png?alt=media\&token=66a20010-fd12-4a18-b8b4-c9a0b8278179)\
  то интеграция уже подключена, а токен можно найти в разделе "Дополнительно".

{% endnote %}

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FJxfDqHcZuqlY1aw9nq9U%2Fimage.png?alt=media&#x26;token=45c5004e-c48d-44d9-bacb-f53abea263d4" alt="" width="563"><figcaption><p>Раздел "Дополнительно", где находится нужный токен</p></figcaption></figure>

4.  Перейдите в Smartbot Pro в раздел "Каналы" и выберите Wazzup<br>

    <figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2Fv9uGVFByaf1GwsmKnn9X%2Fimage.png?alt=media&#x26;token=1de7dd2e-113b-4549-b4e1-33b296cf82ad" alt=""><figcaption></figcaption></figure>
5. Выберите вариант подключения, когда интеграции с CRM нет, и вставляем ключ, полученный ранее в кабинете wazzup

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F1FLpNB6gQlNU3XLA5QWn%2Fimage.png?alt=media&#x26;token=29a162db-eb5a-4498-82fd-127ca099caa2" alt="" width="563"><figcaption></figcaption></figure>

{% note tip %}

Аккаунт Wazzup без интеграции с CRM успешно подключен к Smartbot Pro!

{% endnote %}
