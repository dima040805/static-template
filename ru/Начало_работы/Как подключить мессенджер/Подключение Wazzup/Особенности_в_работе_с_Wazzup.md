# Особенности в работе с Wazzup

При подключении каналов через Wazzup важно учитывать технические ограничения этой платформы, так как она является агрегатором. Эти ограничения могут влиять на функциональность бота. Рассмотрим их подробно для корректной настройки сценариев.

1. **Настройка кнопок имеет некоторые особенности.**\
   В блоке "Отправить сообщение" можно настроить кнопки. Полная функциональность кнопок доступна только для канала WABA (WhatsApp Business). В остальных мессенджерах кнопки не поддерживаются.&#x20;
2. **Функциональность раздела "Чаты" ограничена.**\
   Из раздела "Чаты" нет возможности ответить от имени оператора.
3. **В каналы Wazzup не получиться отправить рассылку.**\
   В Wazzup недоступна массовая рассылка через подключённые каналы мессенджеров.
4. **Бот может писать пользователю в течение 24-х часов с момента последнего сообщения пользователя.**\
   Отправка отложенных сообщений от бота возможна только в течение 24 часов после последней активности пользователя. Это нужно учитывать, если вы планируете настроить сценарий с отложенными таймерами.
5. **Выгрузка пользователей недоступна.**\
   К сожалению, в канале Wazzup нет функции импорта списка пользователей из файла.
