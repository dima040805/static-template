# Подключение других каналов через Wazzup

На данный момент можно подключить и использовать в Smartbot Pro:

1. WhatsApp (личный аккаунт WhatsApp)
2. WABA (WhatsApp Business API, аналог - телеграм бот)
3. Instagram
4. Telegram Personal (личный аккаунт телеграмма)
5. Telegram Bot (обычный телеграм бот)
6. Вконтакте
7. Авито

### WhatsApp

1. Для подключения личного аккаунта WhatsApp зайдите в раздел "Каналы", нажмите "Добавить канал" и в открывшемся окне выберите "WhatsApp"

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FAqYpCVi2cyzM62cg4Zun%2Fimage.png?alt=media&#x26;token=839b7aae-9be9-4d6f-b6f3-75d12ba144ce" alt="" width="563"><figcaption></figcaption></figure>

2. В открывшемся окне также выберите WhatsApp

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FNat2cRIjKiwPJxd6Ky0R%2Fimage.png?alt=media&#x26;token=448756fe-828e-46ee-854d-70a1c8656bcc" alt="" width="563"><figcaption></figcaption></figure>

3. На экране появится инструкция - следуйте ей, чтобы в приложении WhatsApp на телефоне отсканировать QR-код для привязки аккаунта

{% note info %}

Учтите, что при добавлении аккаунта загрузятся 3 последних диалога, остальные будут загружаться по мере поступления сообщений

{% endnote %}

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2Fg8o39FV99vxOfmhjWwWW%2Fimage.png?alt=media&#x26;token=76a574b2-a105-43c1-81dc-93200a3e3efb" alt="" width="563"><figcaption></figcaption></figure>

{% note tip %}

Поздравляем! WhatsApp подключен к вашему аккаунту Wazzup!

{% endnote %}

### Telegram Bot

1. Для подключения бота, созданного в Telegram, зайдите в раздел "Каналы", нажмите "Добавить канал" и выберите Telegram

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FoET1iSTtNMMjlu0UfGkS%2Fimage.png?alt=media&#x26;token=d89b1332-6bb0-4210-8ad4-147956afa874" alt="" width="563"><figcaption></figcaption></figure>

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FAHI3soSx8NnPCK63xwpe%2Fimage.png?alt=media&#x26;token=ea30fdbb-0447-499d-9fc8-5324321619ef" alt="" width="563"><figcaption></figcaption></figure>

Для корректной работы бота убедитесь, что больше он никуда не интегрирован

{% endnote %}

### Вконтакте

1. Для подключения группы Вконтакте зайдите в раздел "Каналы", нажмите "Добавить канал" и выберите Вконтакте

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F1mtgc0Y3QjpXvgPddgOJ%2Fimage.png?alt=media&#x26;token=3af6b6e4-3b52-4c95-8a91-c70034f0006c" alt="" width="563"><figcaption></figcaption></figure>

2. Нажмите кнопку "Авторизоваться" и пройдите авторизацию

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FrOZhR4R7tVQZFVaxgaEn%2Fimage.png?alt=media&#x26;token=e58803ef-90e7-449a-9637-77b186eddd8e" alt="" width="469"><figcaption></figcaption></figure>

3. Выберите группу, которую хотите подключить к аккаунту, и в следующем окне выдайте нужные разрешения

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FU0J73I7zRVzgk1cbXI5r%2Fimage.png?alt=media&#x26;token=4c7970a7-863f-44f9-adc1-7705196167a5" alt="" width="485"><figcaption><p>Выбор группы для подключения</p></figcaption></figure>

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FDde05m3JVoMXvNGBKAZC%2Fimage.png?alt=media&#x26;token=69d710b0-3269-4d74-95a9-c1a437c885b1" alt="" width="563"><figcaption><p>Выдача разрешений</p></figcaption></figure>

4. Из этого же окна вам нужно будет скопировать адрес строки, для того чтобы подключить группу к Wazzup, и вставить в окно с названием "Добавление канала" и нажать "Добавить"

{% note warning %}

Wazzup предупреждает, что использовать данные из адресной строки для сторонних сайтов не стоит, т.к. можно потерять доступ к своему аккаунту!

{% endnote %}

{% note tip %}

Поздравляем! Вконтакте подключен к вашему аккаунту Wazzup!

{% endnote %}
