# Добавление бота в канал Telegram

Для того, чтобы добавить бота в канал, достаточно сделать его администратором в канале. Это может понадобится, например, для сбора информации о подписчиках.

{% note info %}

Если у вашего канала есть комментарии и нужно, чтобы бот отвечал в них, добавьте бота в групповой чат, который привязан к каналу, [по инструкции](https://docs.smartbotpro.ru/nachalo-raboty/kak-podklyuchit-messendzher/podklyuchenie-telegram/dobavlenie-bota-v-gruppovoi-chat-v-telegram).

{% endnote %}

Добавьте бота в канал в качестве участника и вам будет предложено сразу сделать его администратором

**Обратите внимание**: если ваш телеграм-канал не отображается в блоке проверки подписки, убедитесь, что вы лично писали в вашего бота или хотя бы запускали его. У вас с ботом должен быть открытый диалог.

{% note tip %}

Отлично! Теперь бот добавлен в ваш канал!
