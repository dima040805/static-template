# Особенности в работе с MAX

На данный момент есть определенные ограничения в работе канала MAX. Как только что-то изменится, мы это поддержим и отразим в этой документации.

1.  **Особенности загрузки пользователей**\
    Обычно загрузка базы позволяет восстановить пользователей, которые ранее писали в бота до подключения его к Smartbot Pro. После загрузки бот может отправлять им сообщения, так как система проверяет наличие разрешения на общение.<br>

    Для канала **MAX** есть особенность: при загрузке базы такая проверка не выполняется. Если пользователь написал боту, а затем заблокировал его, система этого не отразит. Поэтому после загрузки рекомендуется дополнительно проверить, кому из пользователей можно отправлять сообщения.
2.  **Поддерживаются только inline-кнопки**

    При отправке кнопки другого типа вы увидите в отладочном сообщении ошибку, которая уведомит вас о невозможности использовать такой вид кнопок.
3.  **Белый цвет кнопок недоступен**\
    У кнопок можно выбрать 4 цвета - синий, зеленый, красный и белый, но если белую кнопку отправить в MAX, то она просто пропадет из списка.<br>

4.  **Разметка «спойлер» не поддерживается**\
    При выборе такой разметки для текста в МАХ она исчезнет с текста будто ее и не было<br>

5. **Вложения типа «стикер» не загружаются**\
   Такие вложения никак нельзя загрузить для дальнейшей отправки или обработки ботом.
6. **Особенности работы с настройкой «Не отправлять уведомление»**\
   В блоке есть возможность не отправлять уведомление при отправке сообщения и этот функционал работает только в групповом чате. Если сообщение с такой галкой будет отправлено в ЛС с ботом, уведомление все равно прийдет.
7. **Блок "Проверка подписки" недоступен**\
   Функция проверки подписки для пользователей в канале MAX не работает.
8. **Удаление последнего сообщения и временное удаление не поддерживаются**\
   В блоке **«**&#x423;далить пользователя из чата» для канала MAX доступно только удаление пользователя навсегда. Удаление последнего сообщения и временное удаление недоступны.
9. **Рассылки в каналы MAX не отправляются**\
   Массовые рассылки недоступны для этого канала.
10. **Меню команд для бота недоступно**\
    Пока что нет функционала собрать список команд для бота, поэтому учитывайте это при сборке сценария.
