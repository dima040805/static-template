# Регистрация и личный кабинет


<p class="yfm-lead">Перейдите по адресу smartbotpro.ru. Здесь вы можете почитать про сервис:</p>
Чтобы попасть на страницу авторизации, нажмите на кнопку « Попробовать бесплатно»:

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FNn0r7ws9yJJEI2IofHOU%2Fimage.png?alt=media&#x26;token=077de8f1-28c0-4008-ac50-30081384d361" alt=""><figcaption></figcaption></figure>

{% note info %}

После регистрации начнется бесплатный тестовый период, который действует 1 неделю. \
В это время вам будут доступны все возможности сервиса.

{% endnote %}

Нужно зарегистрировать кабинет. Здесь будут все ваши сценарии и проекты. Если вы, например, маркетинговое агентство, вы можете делать отдельные проекты под каждого клиента, и они не будут пересекаться.

Для регистрации кабинета введите имя, почту и пароль (или выберите другой способ авторизации: через google, ВКонтакте, Telegram, Elama) и нажмите «Продолжить»:

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FtcGjAZ7FUBhGLLCciGwK%2Fimage.png?alt=media&#x26;token=733d5c99-34ff-4928-b343-b40cf6d9c1eb" alt=""><figcaption></figcaption></figure>

Далее вводим название вашей компании (далее - название вашего кабинета) и выбираем удобный поддомен (начальный вариант будет предложен автоматически).&#x20;

{% note info %}

Поле "Название компании" необязательно и если на момент создания кабинета оно будет пустым, то поддомен кабинета будет создан автоматически из вашей почты.

{% endnote %}

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2F1KedSJW4qrE7K7GTX0Us%2Fimage.png?alt=media&#x26;token=987aafc8-660b-47b6-887f-a9336a7b4e66" alt=""><figcaption></figcaption></figure>

После этого мы сразу попадаем в свой кабинет.

В адресной строке в этот момент уже стоит выбранный домен. Теперь вы можете попадать в конструктор по этому адресу. <br>

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FNltdZroZL4tvKZsUtl2S%2Fimage.png?alt=media&#x26;token=e972499c-e88d-4b1c-8cb6-fcbcfef2f7b3" alt=""><figcaption></figcaption></figure>

Посмотрите: сверху, если развернуть список кабинетов, будут видны все, по которым вы уже подключались. Сейчас там один кабинет — это KTS\_smarttest.

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FqUtc3Zz0U4jjJZjFqMdA%2Fimage.png?alt=media&#x26;token=be24fd45-3903-49ba-9f16-ca8443e54efb" alt=""><figcaption></figcaption></figure>

Изначально создан один основной проект, но вы можете создать и другие. В рамках кабинета их может быть несколько, оплата будет общей с баланса кабинета.&#x20;

Для создания проекта кликните по иконке профиля, разверните список проектов и нажмите кнопку "Создать проект".

<figure><img src="https://4058588211-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvTd8XmFMqkqZga7zhLuk%2Fuploads%2FE44n0VymQSY1XLlQficW%2Fimage.png?alt=media&#x26;token=f4fd98ad-3cf9-40bc-86ef-06f50ead93fa" alt=""><figcaption><p>При клике по иконке профиля можно увидеть список доступных кабинетов и проектов, перейти в настройки профиля и выйти из аккаунта</p></figcaption></figure>

### Как поменять почту своего аккаунта

Если вы зарегистрировались как Владелец кабинета под неверной почтой, то просто [добавьте себя в качестве нового владельца кабинета](../nachalo-raboty/dobavlenie-menedzherov) через раздел настроек. Потом зайдите под нужной почтой и удалите свой старый аккаунт менеджера.&#x20;
